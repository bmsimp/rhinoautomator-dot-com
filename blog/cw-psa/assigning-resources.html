// BLOG — CONNECTWISE PSA ACTIONS

ASSIGNING RESOURCES
TO TICKETS

REQUIRED VARIABLES

  • ticket_id — The ID of the ticket you are assigning the member to
  • member_id — The ID of the member you are assigning to the ticket

ACTION

POST

URL PATH

{ base URL including version identifier }/schedule/entries

PAYLOAD

{{
    {
    "objectId": CTX.ticket_id|int,
    "member": {
        "id": CTX.member_id|int
        },
    "type": {
        "id": 4
        }
    }
}}
