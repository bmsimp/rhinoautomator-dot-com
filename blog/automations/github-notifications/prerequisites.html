PREREQUISITES

Before you get started, you'll need the following things:

  • A GitHub account. If you do not already have one, you can sign up here.
  • A GitHub repository that you want to track changes. This example uses RewstApp/docs.rewst.help.
  • The Pull GitHub App open in a tab in your browser.
  • Access to your preferred automation platform of choice. I'll be using Rewst, but you can adapt this to your preferred platform. If your platform is capable of receiving webhooks, this will provide you with the best flexibility in the information provided.
