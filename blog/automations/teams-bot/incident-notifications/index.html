SYSTEM INCIDENT
MAINTENANCE NOTIFICATIONS

We have so many tools that we use on a day-to-day basis that can have an impact if they experience an outage. It's helpful for everyone to be aware of this while not clogging up email inboxes with notifications (often several per incident). I wanted to build out a system that would send the initial notice to Teams and then update that same card as updates rolled in.
